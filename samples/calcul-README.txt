How to Launch sample:
	- Web:
	   - Unix:
	   Type command : 'java -jar jac.jar -R . -C src:samples.jar -W
	   default src/org/objectweb/jac/samples/calcul/calcul.jac' in the jac root
	   directory, to launch the program with the web GUI.

	   - Windows:
	   Double-click on 'calcul-web.lnk' to launch the web GUI.

	You must then open a web browser and connect on the URL :
	'http://localhost:8088/org/objectweb/jac/default', where 'localhost' is
	your ip.
	
	- Swing:
	   - Unix:
	   Type command : 'java -jar jac.jar -R . -C src:samples.jar -G
	   default src/org/objectweb/jac/samples/calcul/calcul.jac' in the jac root
	   directory, to launch the program with the swing GUI.

	   - Windows:
	   Double-click on 'calcul-swing.lnk' to launch the Swing GUI.

Description: 
	This sample is a just a simple calculator, with add and sub
	methods, but with lots of aspects applied to it.
	To see which aspects are loaded, open calcul.jac.
